# 🏊‍♂️ Кубок ВС РФ по плаванию

Профессиональное приложение для подсчета командных результатов соревнований по плаванию.

## 🚀 Быстрый старт
1. Запустите `SwimmingCupApp.exe`
2. Следуйте инструкциям в приложении

## 📁 Структура проекта
- `swimming_cup_app.py` - основной код
- `build_exe.py` - скрипт сборки
- `dist/SwimmingCupApp.exe` - готовое приложение
